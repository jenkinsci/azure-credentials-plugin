Leave it blank if you are using the public cloud environment chosen above. Otherwise if you are using private Azure cloud service, choose a base environment from the above and then override the active directory endpoint URL.

The active directory endpoint URL's used in each of the environments are:

  • Azure China - https://login.chinacloudapi.cn/

  • Azure US Governmant - https://login-us.microsoftonline.com/
