Leave it blank if you are using the public cloud environment chosen above. Otherwise if you are using private Azure cloud service, choose a base environment from the above and then override the management endpoint URL.

The management endpoint URL's used in each of the environments are:
  • Azure - https://management.core.windows.net/
  • Azure China - https://management.core.chinacloudapi.cn/
  • Azure Germany - https://management.core.cloudapi.de/
  • Azure US Governmant - https://management.core.usgovcloudapi.net/
